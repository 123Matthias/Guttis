Datenschutzerklärung

Wir nehmen den Schutz Ihrer persönlichen Daten sehr ernst. Diese Datenschutzerklärung informiert Sie darüber, wie wir mit Ihren Daten umgehen, wenn Sie unsere Website besuchen.

1. Erhebung und Verarbeitung von Daten

Unsere Website erhebt keine persönlichen Daten. Bei Nutzung von externen Diensten wie Bootstrap können jedoch Daten, wie z.B. Ihre IP-Adresse, erfasst werden. Diese Daten werden von den jeweiligen Anbietern verarbeitet.

2. Nutzung von Cookies

Unsere Website verwendet keine eigenen Cookies. Es werden jedoch Cookies von externen Diensten gesetzt, wie z.B. von Bootstrap (für die Darstellung von Schriftarten und anderen Ressourcen).

3. Drittanbieter

Unsere Website nutzt externe Dienste wie Bootstrap, die Daten wie Ihre IP-Adresse sammeln können. Wir haben keinen Einfluss auf diese Datenerhebung. Weitere Informationen finden Sie in den Datenschutzerklärungen von: Bootstrap

4. Externe Dienste

Unsere Website wird über GitHub Pages gehostet und nutzt den Proxy-Service von Cloudflare für bessere Performance und Sicherheit. Zudem wurde die Domain über CheckDomain registriert. Diese Dienste können Daten wie IP-Adressen verarbeiten. Weitere Informationen finden Sie in den Datenschutzerklärungen von GitHub Cloudflare und CheckDomain .

Wir behalten uns vor, diese Datenschutzerklärung jederzeit anzupassen. Die aktuelle Version ist immer auf dieser Seite verfügbar.
